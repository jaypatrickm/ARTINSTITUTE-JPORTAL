Dinosaurus - 2010 -- www.dinosaurus.com

Challenge

Design a website for teenagers to use for learning about dinosaurs. The site must make use of javascript and jquery.

Solution

Presented a website with a jungle colored them reflective of dinosaur environments. Use of a hover navigation, slideshow, random item generator, tabs, accordion, greybox, form and validation. Project

Volcano - 2010 -- www.volcano.com -- Before

www.volcano.com -- After

Challenge

Redesign a website created in a class contest: to create a javascript/jquery website in four hours.

Solution

The previous design resulted in flaws regarding how the website was read (left to right), repetitive links, and a dated design aesthetic. I did away with the defining red line that confused the reading. The links were remade into one global navigation. I edited the color scheme and graphics to give it a younger and more modern look. Research on teenage aimed websites was done and resulted in images with text over it, smaller groups of text, but in a larger size.

Project

The Vinylmation Station - 2011 -- www.thevinylmationstation.com

Challenge

Create a website utilizing some kind of flash or actionscript

Solution

Designing a website completely in flash brings up a number of problems with search engine optimization, multiple devices, and load times. It was better to incorporate a part of your website with some of the animation. The Vinylmation station is a website for Disney's latest collectable trend. The target audience is all ages especially Disney lovers. I used a Mickey Mouse color scheme to immediately give a sense of Disney. I utilized actionscript to make hover state series gallery.
